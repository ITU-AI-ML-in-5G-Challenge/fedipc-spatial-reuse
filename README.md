# fedipc-spatial-reuse
FedIPC Spatial Reuse Project for ITU AI/ML Challenge 2021.


## Setup
* `pip install requirements.txt`

## Running
* `python main.py`
* or `python main.py --preprocessor all_features` 


## Parameters
* `--preprocessor`
    * `basic_features`
* `--model`
    * `fed_avg`
* `--metrics`
    * `mse`
